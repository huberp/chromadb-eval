About Browser-based Vector Search

A standalone browser demo of semantic search using vector embeddings. All processing happens in your browser with no backend or database required.

üîß Technologies

  • Transformers.js: Browser-native ML via @huggingface/transformers
  • Embedding Model: Xenova/all-mpnet-base-v2 (768-dim vectors)
  • In-Memory Search: JavaScript-based cosine similarity
  • Static Hosting: GitHub Pages

üì¶ Embedding Preparation

  • Structure-based chunking of Markdown files using AST parsing
  • Semantic sections preserved with configurable chunk size & overlap
  • Pre-computed embeddings generated via GitHub Actions workflow
  • Stored on GitHub in the data-main branch

‚öôÔ∏è How It Works

  • Pre-computed embeddings loaded from GitHub at runtime
  • Your query is embedded in-browser using Transformers.js
  • JavaScript computes cosine similarity to find top matches
  • No server, no database - everything runs locally
